Recipe Gallery Application

This project is a Recipe Gallery developed using React with Vite. The purpose of this assignment is to demonstrate an understanding of rendering static data in React using the map() function, without implementing state management or user interactivity.

The application displays a curated list of recipes stored in a constant array. Each recipe includes an id, title, image, and a list of ingredients. The recipes are rendered as individual cards using React’s list rendering pattern, and each card is assigned a unique key based on the recipe id to follow React best practices.

The user interface is styled using CSS and organized with a responsive grid layout to ensure the gallery displays correctly across different screen sizes. This project focuses on clean component structure, readability, and proper React fundamentals.

How to Run the Application

Ensure Node.js is installed on your system.

Open a terminal and navigate to the project directory:

cd recipe-gallery

Install all required dependencies:

npm install

Start the development server:

npm run dev

Open a web browser and navigate to the local server address provided in the terminal (typically http://localhost:5173).
